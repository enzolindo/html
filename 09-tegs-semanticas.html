Tags semânticas

As tags semanticas são elementos HIML que descrevem claramente o significado do conteudo que elas envolvem. Elas ajudam a melhorer a acessibilidade, a SEO e a estrutura do documento. Aqui estão algumas das principais tags semánticas:

  • <header>: Define o cabeçalho de um documento ou seção.
  • <navigt;: Define um conjunto de links de navegação
  • <main>: Define o conteúdo principal do documento.
  • <sectionsgt;: Define uma seção genérica de um documento.
  • <article>: Define um conteúdo independente e auto-contido, como um artigo de blog ou noticia.
  • <aside>: Define un conteúdo relacionado, mas separado do conteúdo principal, como una barra lateral.
  • <footer> : Define o rodapé de um documento ou seção.

    • Usar essas tags semánticas ajuda os motores de busca a entender melhor o conteúdo do seu site e melhora a experiência do usuário, especialmente para aqueles que utilizam leitores de tela.
